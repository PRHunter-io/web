# [PRHunter.io](https://prhunter.io) web app

PRHunter is a web3 platform that levarages smart contracts to facilitate payments for Github Pull Requests. The app consists of:

- A Next.js web app that users interact with (this repo)
- A Kotlin/Spring Boot backend that supports the platform (not public yet)
- Smart contracts deployed onto BSC and ETH [PRHunter-io/bounty-contracts](https://github.com/PRHunter-io/bounty-contracts)

## Feedback on the platform

If there's any bugs you'd like to report just open an issue on this repository. For features requests, please use the discussions tab.

## Usage

You can run the app locally, by default it's interacting with our test instance over at `staging.prhunter.io`
